# StarLight Frontend

Angular 15 application for the StarLight blogging platform.

See the [main README](../README.md) for full setup, deployment, and API documentation.

## Quick start

```bash
npm install
npm start
```

Opens **http://localhost:4200** with API requests proxied to `http://localhost:8080`.

## Scripts

| Script | Description |
|--------|-------------|
| `npm start` | Dev server with API proxy |
| `npm run build:prod` | Production build |
| `npm test` | Karma unit tests |
